API Integration Information

What This Does

API integration allows you to automatically send your brand voice parameters to external AI copywriting systems. This enables these systems to generate content that matches your brand's unique voice.

When enabled, your brand voice parameters will be sent to the selected API provider whenever you update them through any of the input methods.

Security Information
  • Your API key is stored only in your session and is not saved to any database.
  • The connection to the API provider is made using secure HTTPS.
  • We recommend using API keys with limited permissions when possible.
  • You can disable the integration at any time.
